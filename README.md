## Usage

No build step required. Just open `index.html` in a browser, or deploy directly to GitHub Pages.

```
ユーザー名.github.io
└── index.html   ← このファイルがすべて
└── README.md
└── .gitignore
```

## Customization

Edit `index.html` directly. Key sections to update:

- **Name / title** — `<title>` tag and `.hero-name`
- **SNS links** — Contact page, `<a class="link-item" href="#">` の `href` を差し替え
- **Blog posts** — `.blog-post` ブロックを複製・編集
- **Awards** — `.award-item` ブロックを追加
